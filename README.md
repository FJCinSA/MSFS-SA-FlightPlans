# MSFS 2024 — Southern Africa Flight Pack
**CoetzeeMed Flight Operations · FJCinSA**

Flight plans for Microsoft Flight Simulator 2024 covering Southern Africa.
Built for realism — real SA AIP navaids, VOR/NDB raw navigation, authentic routes.

## What's In This Pack

### Personal History Flights — ZS-NTP Recreation
*Real flights flown by Dr Francois Coetzee in ZS-NTP, a 1978 Cessna 172N based at Wonderboom.*
*Raw navigation — steam gauges, ADF, VOR. No GPS. No autopilot. Just the needles and the clock.*

**Wonderboom → Mafikeng via Pilanesberg**
James Caroto-Coetzee's first ever flight. Comms failure into Pilanesberg — relayed via Johannesburg Information. James handled the cellphone, Francois flew the aircraft. Midsummer, thermal turbulence, bumpy ride.

| Leg | Route | Notes |
|-----|-------|-------|
| FAWB → FAPN | via WB NDB / Hartbeespoort / PLB NDB / PNV VOR-DME | Comms failure this leg |
| FAPN → FAMM | via PNV / Zeerust / MMV VOR-DME / MH NDB | |

**Wonderboom → Stellenbosch via Karoo**
Solo. Night flight over the Karoo — dark moon, no flight plan, 8000ft, scattered farmhouse lights the only ground reference. Cloud base on the mountains forced a divert to Worcester. Flew Du Toitskloof under the cloud, mountains visible on both wings, following the N1.

| Leg | Route | Notes |
|-----|-------|-------|
| FAWB → FATM | via WB NDB / PSV VOR-DME / BLV VORTAC | Avoided FABL fees, parked at Tempe |
| FATM → FABW | via BLV / VWV VOR | Night. Dark moon. Trust the instruments. |
| FABW → FAWC | via SWV VOR / Hex River Valley | Cloud base dropping. Diverted Worcester. |
| FAWC → FASH | via Du Toitskloof Pass | Low level. Mountains both sides. Afternoon light. |

## How To Use

**MSFS 2024:** World Map → Load → browse to `C:\FlightPlans\` → select a `.pln`

**Little Navmap:** File → Open Flight Plan → select a `.pln`. Use the navaid pages to tune VORs and NDBs alongside the map.
