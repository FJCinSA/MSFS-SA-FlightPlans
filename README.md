# MSFS 2024 — Southern Africa Flight Plans

Flight plans for Microsoft Flight Simulator 2024 covering Southern Africa routes.
Built for realism — VOR/NDB raw navigation, accurate SA navaids, forum-quality plans.

## Routes

### FACT → FALA (Cape Town to Lanseria)
Three-leg IFR route via Beaufort West and Colesberg.
Each leg available in GPS edition and RAW NAV edition (VOR/NDB/dead reckoning).

| Leg | Route | Distance | Aircraft |
|-----|-------|----------|---------|
| Leg 1 | FACT → FABW | ~230 nm | C172 |
| Leg 2 | FABW → FACB | ~195 nm | C172 |
| Leg 3a | FACB → FABL | ~195 nm | C172 |
| Leg 3b | FABL → FALA | ~235 nm | C172 |

## Navigation
RAW NAV editions use real South African AIP navaids:
BDV VOR 113.1MHz — BL NDB 380KHz — BLV VORTAC 114.1MHz — KS NDB 228KHz — LIV VOR-DME 117.4MHz

## Files
- GPS editions: direct airport-to-airport, for G1000 aircraft
- RAW editions: VOR/NDB/dead reckoning, for steam gauge aircraft
- Kneeboards: HTML reference cards with headings, frequencies, timing
