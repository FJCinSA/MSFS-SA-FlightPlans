Using This Pack

MSFS 2024

World Map → Load → browse to C:\FlightPlans\ and pick a .pln. Waypoints load as the planned route.

Little Navmap

File → Open Flight Plan… → select a .pln. Use the navaid pages alongside the map to tune VORs & NDBs.

Reference pages

The three HTML files are self-contained — double-click to open in any browser, no internet needed. Search works offline.
